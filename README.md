# div-wang的博客

div-wang的技术随想录😄

## 如何开始

请修改 `src` 目录里的对应文件

* 本地有 Node.JS 和 Python 2.7 环境;  
* 全局安装了 gulp，如果没有请 `npm i -g gulp`;  
* 克隆本项目`https://github.com/div-wang/div-wang.github.io.git`; 
* 进入项目目录执行 `npm install`;   
* 终端输入 `gulp` 即可开始  

Project page: http://div-wang.github.io
